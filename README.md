## README

This README will document whatever steps are necessary to get **smoke-signals** up and running.

### Setup

* `git clone <repo-url>`
* `bundle install`
* `rails s`
* `rackup faye.ru -s thin -E production`
* `redis-server`
* Visit `http://localhost:3000` in your browser
